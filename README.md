### 💭 **배경**

- 과거 실버세대와 달라진 5060 뉴시니어 세대는 여유 있는 자산을 기반으로 자신의 인생을 누리는데 기꺼이 시간과 비용을 투자함.
- 뉴시니어 세대는 소비시장의 새로운 중심으로 성장하고 있음.
